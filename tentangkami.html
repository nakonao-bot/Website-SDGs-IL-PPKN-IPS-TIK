Profil Saya

Naoko Pema

Kelas: IX-2 | No. Absen: 26

Tujuan Website

Tujuan pembuatan website ini adalah untuk memenuhi nilai SAS TIK, ulangan PPKn, dan tugas IPS. Website ini berisikan informasi mengenai SDGs, terutama SDG 5: Kesetaraan Gender, serta bentuk kerja sama internasional yang dapat mendukung pencapaian tujuan tersebut. Selain itu, website ini juga bertujuan untuk meningkatkan pemahaman saya tentang pentingnya kesetaraan gender dan kerja sama global untuk mempercepat tercapainya tujuan tersebut. Dengan membuat website ini, saya berharap dapat berbagi pengetahuan dan meningkatkan kesadaran tentang isu-isu penting yang dihadapi dunia saat ini. Saya juga ingin membantu masyarakat memahami bagaimana mereka dapat berkontribusi dalam mencapai SDGs, khususnya dalam hal kesetaraan gender, karena dalam beberapa tahun kebelakang, isu ini semakin banyak saya dengar di berbagai media. Sehingga, saya berharap website ini dapat memberikan manfaat bagi pengunjung.

Refleksi

Melalui pembuatan website ini, saya belajar banyak tentang pentingnya kesetaraan gender dalam pembangunan berkelanjutan. Saya menyadari bahwa kesetaraan gender bukan hanya masalah perempuan, tetapi juga melibatkan laki-laki dan seluruh masyarakat. Saya juga memahami bahwa kerja sama internasional sangat penting untuk mencapai tujuan global seperti SDGs. Selain itu, saya juga belajar tentang berbagai tantangan yang dihadapi dalam mencapai kesetaraan gender, seperti stereotip sosial, diskriminasi, dan kekerasan berbasis gender. Saya menyadari bahwa perubahan budaya dan sikap masyarakat sangat diperlukan untuk mengatasi tantangan tersebut. Saya merasa bersyukur untuk dapat belajar dan berkontribusi dalam isu penting ini melalui pembuatan website. Saya berharap pengetahuan yang saya peroleh dapat saya terapkan dalam kehidupan sehari-hari dan saya dapat menjadi agen perubahan untuk mempromosikan kesetaraan gender di lingkungan sekitar saya. Saya juga bersyukur karena saya menyadari bahwa saya merasakan kesetaraan gender di kehidupan saya, yaitu saya dapat kesempatan untuk belajar, berpendapat, dan mengejar cita-cita tanpa adanya diskriminasi berdasarkan gender. Hal ini membuat saya semakin termotivasi untuk memperjuangkan kesetaraan gender bagi semua orang, walaupun sebagai pelajar. Saya juga sangat berterima kasih kepada pahlawan perempuan yang telah berjuang untuk hak-hak perempuan di masa lalu, terutama di Indonesia, sehingga saya dan banyak perempuan lainnya dapat merasakan kesetaraan gender saat ini, seperti R.A. Kartini, Cut Nyak Dien, dan yang lainnya. Sehingga, saya berharap website ini dapat memberikan manfaat bagi pengunjung dan meningkatkan kesadaran tentang pentingnya kesetaraan gender.
